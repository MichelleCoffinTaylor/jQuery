JQuery Introduction

JQuery is an open source Javascript Library. It greatly simplifies javascript programming and has hundreds to pre built functions which makes writing javascript alot easier.
The reason we are learning jQuery before we do javascript is that it is a lot easier to understand and will help us learn Javascript easier in the future.
When we use jQuery, we will be manipulate the DOM (Document Object Model). The DOM is in reference to every element in your page.

Installation

There are two main ways to get jQuery into your website. You can download the full jQuery library from the jQuery Wesbite. Or you can use a CDN

Download Library

If you want to have a local version of you you need to download the full library from the jQuery Wesbite

There are two different versions of the jQuery you can download

  • Production
  • Development

Development should be used during your development stage and when you go live you should switch it to production which is more compressed with a smaller file size

Pros of downloading library
  • You have a local version of jQuery in your project so you can use jQuery when you aren't connected to the internet.
  • Loading times will be quite fast as you have it in your project.
Cons of using download library
  • It takes up some disk space on your live server which could be given to something else.
  • Your local jQuery won't be up to date if a new version comes out.

Content Delivery Network (CDN)

If you don't want to include a local version you can use a cdn from an alraedy exsisting online serve.Google has a wide range of different hosted libraries which you can use/

Pros of using a CDN
  • The CDN will always be the latest version of the library.
  • Many internet users already downloaded jQuery using a CDN when they have visted another site using the same CDN. This means it will be loaded into their cache so when they visit your site it will load from their cache instead of the host, which will be much faster.
Cons of using a CDN
  • If for some reason the CDN is down and your users haven't used jQuery before, any functions called won't be able to run
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.0/jquery.min.js"></script>

Your script link to the jQuery should be put at the bottom of your body. Sometimes javascript and jQuery can take quite awhile to load. Because HTML/CSS renders from top to bottom. This means that all the elements on your page will be loaded first and then it will start to run the jQuery. You then want to create your own javascript file for your project. This should be called after you call jQuery. If you call it before hand, the scripts won't recognise any of the jQuery functions you use.

Document Ready

jQuery is made to help up select and manipluate elements easy. You can select element tags, id's or classes. The basic syntax for writing jQuery is $(selector).action();. The $ is calling access to jQuery. Selector is the element which you are selecting. and action is the action which is being performed.
jQuery has a bunch of pre build actions which would usually take us serveal lines of code to get working but only need one word.

Getting the document ready

When writing jQuery you should write it within the following function

$(document).ready(function(){

});

Adding it within the document ready function will prevent jQuery from running in the document until the page is finished loading.

Selecting Elements

In jQuery you can select many different elements. You select them quite similar to CSS

Elements

$("p")

This selector will select all of the p elements within the page.

ID's

$("#id")

This selector will select the element on the page with the ID given.

Classes

$(".class")

This selector will select all of the elements which have the given class.

This

$(this)

$(this) is a selector which can only be used when a jQuery function has occurred. It is in reference to the element with triggers the event. Read the example bellow.

$("#Example").click(function(){
	$(this).hide();
});

What is happening in this example is when a element with the ID of Example is clicked we want to hide that element. Instead of adding $('#Example').hide(); we can just say $(this). In this example $(this) means $('#Example') beacuse that is the element which is calling the function.

Common jQuery Events

jQuery has a wide range of pre built events which get activated when specific events happen in the page. These functions are called within the document ready function.
Bellow are some of the common functions which might get used

  • Click
  • Double click
  • Hover
  • Mouse Enter
  • Mouse Leave

Click

The click event gets activated when the user clicks on the trigger element.

$("#Test").click(function(){

});

What this is saying is when someone clicks on the element with the ID of Test, to run the function which would be written within the function. Click on the example bellow to get an alert pop up.

Double Click

The Double Click event is very similar to the click event, but the only difference is that you needs to double click on the element.

$("#Test").dblclick(function(){

});

What this is saying is when someone Double Clicks on the element with the ID of Test, to run the function which would be written within the function. Double click on the button bellow to get an alert pop up. If you do a single click on it then nothing will happen.

Mouse Enter

The mouse enter event get triggered when you cursor is over the element.

$("#Test").mouseover(function(){

});

Mouse Leave

The mouse leave event gets triggered when your cursor leave the element which you have entered.

$("#Test").mouseover(function(){

});

Hover

The Hover event is a mix of both the mouse enter and leave function. Functions which you run within this function will run when your mouse enters the element and stops when you leave it. Hover is an event which you could do in CSS to change properties of the element. But in jQuery we can use it to run other jQuery functions. This event gets called when the cursor is over the element.

$("#Test").hover(function(){

});

In the Example bellow, when you hover over the button, another one should appear next to it.

There are many other pre built events which you can run using jQuery.

  • mousedown()
  • mouseup()
  • focus()
  • blur()
  • on()

Effects

Along with events, jQuery has a bunch of pre built effects which can help us out with our coding.

Many html elements have pre built events which happen. An example is the <a> tag. When you click on it, if there is a href then you will be directed to the url which is described. The prevent default function stops the default event from running, we can then call other things we want to happen instead.

event.preventDefault();

Hide and Show

The hide function is used for hiding an element on the page. What it does it adds Dispay:none to the element

$("Example").hide();

The show function adds Display:block back onto the the element.

$("Example").show();

Bellow is an example which hides a box when you click the hide button, but shows it again when you click the show button.

You can also set the amount of time for the hide and show function to run. In the example bellow it states that the show function would take 5 seconds for the element to fully show.

$("Example").show(5000);

Toggle

Toggle is a function which does the hide and show function together. It satates if the element has a display of block, then when the event happens, turn that into display none. And if it is display none when the event gets called, give it a display of block

$("Example").toggle();

Just like the hide and show functions you can add speed to it the exact same way as teh one above.

Fade In & Fade Out

This is once again similar to the above effects. Just like hide and show it will change its display property depending on what effect is used. It will also change the opacity of the object and change that value depending on which effect you use. It slowly either increases or decreases the opacity until it is either 0 for fade out and 1 for fade in.

$("Example").fadeIn();
$("Example").fadeOut();

Once again there is a wide range of other effects which we could add to elements and these are only a sample.

Manipulating the DOM

So far we have only been using jQuery to either create events or hiding or showing elements. One of the best things about javascript is that it gives us the ability to completly change the DOM. We can add elements to the page which weren't originally in the html. We can remove elements, empty entire sections and much more.

Append

Append is a jQuery function for adding objects into the DOM. Append will add whatever you set to the end of the parent you specify. You should always append something to a parent element as if you don't then it won't know what you are wanting to do.

$("p").append("Some random Text to the end");

What the line of code above says, add the string "Some Random Text" to the end of all <p> tags you have in the site. If you have a button which always appends the line then it will constantly be adding it and will never stop.

This is the starting text.

Using Prepend you can also add html elements into the page. Instead of just writing a normal string, you can write out a full html element and that will append into the parent. An Example of this is if you had a <ul> element and you wanted to keep adding <li> elements into the list.

$("ul").append("<li>This is a new list item</li>");

  • This is the original list item

Prepend

Prepend works the exact same as the append function. The only difference is that it add's it to the beginning of the parent element.

$("p").prepend("Some random Text to the beginning");

This is the starting text.

Remove

Remove is a function which completely removes html elements from the pages. When this function is called, the browser will render your html file but will completely leave out what you have asked to remove in jQuery.

$("#RemoveExample").remove();

In the code above, it is completely removing the element with the ID of RemoveExample. In the example bellow, if you inspect the page and find where we have <div id="RemoveExample"></div>. When you click the remove element button, this div should be removed.

Empty

Sometimes you might not want to completely remove the element, but rather just empty all of its contents. The empty function will empty everything within the selected element.

$("#EmptyExample").empty();

This line of code will remove all of the child elements the selected div but it won't affect itself.

Add and Remove Class

jQuery gives us a real simple way to add and remove classes to our html elements. When you use addClass, if that class doesn't already exsist on the element then it will add it, if you use removeClass, if it exsists on the element it will remove it.

$("#ID").addClass('Red');
$("#ID").removeClass('Blue');

What this states is that it will add the class 'red' to the element with the ID of ID and it will also remove the class 'blue' from that element aswell.

What this example is doing is when you click add Red, it removes the class blue from the element and adds the class red. When you click the add blue button it does the opotsite.

Toggle Class

Toggle Class is the same as Add/Remove Class, only it simplifies the function to include both. If you are toggling a class. If it already exsits on an element, it will remove it, and if it doesn't it will add it.

$("#ID").toggleClass('Large');

This line of code above states that it is toggling a class called large on and off.

Clicking the toggle size button will add on the class of large which has a width of 600px. WHen you click it again it will remove that class and then it will revert to its original size.

CSS

So far we have been talking about manipulating the DOM using classes. What we can do is directly change the css of an element using jQuery.

$("#ID").css('background-color', 'yellow');

This adds just a single CSS style to the element. It won't change anything in the exsisting CSS file, but it will add style="" directly into the html element and because we are adding it to the element itself, that value will overide any exsisting values which might conflict with it.
You can also change several different css elements this way by putting the values into an array.

$("#ID").css({'background-color':'yellow','width':'600px'})

Clicking the add CSS button will directly add on a background colour of yellow and a width of 600px to the element. Originally they have speicifed background colour and width values already but this will overide those values.
